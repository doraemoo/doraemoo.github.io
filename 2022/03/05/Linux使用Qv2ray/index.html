Linux使用Qv2ray

Linux安装Qv2ray教程

以kali为例 首先需要snap包管理软件 不会装的自行百度

执行以下命令下载Qv2ray

snap install qv2ray

安装成功后 下载V2Ray Core

官方下载链接：https://github.com/v2fly/v2ray-core/releases/download/v4.44.0/v2ray-linux-64.zip

wget https://github.com/v2fly/v2ray-core/releases/download/v4.44.0/v2ray-linux-64.zip

下载后解压缩

unzip v2ray-linux-64.zip

完成上述步骤后启动qv2ray

更改v2ray核心可执行文件为解压出来的v2ray文件

v2ray资源目录设置为v2ray的目录，设置完成后点击检查

出现如下信息就说明成功

设置分组信息及订阅地址 再更新订阅就可以使用了
